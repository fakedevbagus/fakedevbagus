Features

🎬 Download Engine

  • Multi-platform support (YouTube, TikTok, Facebook, and more via yt-dlp)
  • Smart resolution and format detection before download
  • High-quality video up to 4K resolution
  • Real-time download progress tracking via SSE
  • FIFO queue system with concurrent download management

🎵 Audio Processing

  • High-quality audio extraction (FLAC, WAV, MP3, M4A, Opus)
  • Playlist download with optional FLAC merge
  • Optional high compatibility mode (H.264/AAC re-encode)
  • Automatic format conversion via FFmpeg

🎨 Interface & UX

  • Dynamic UI themes based on detected platform
  • Glassmorphism design with animated glow effects
  • Dark / Light mode toggle
  • Responsive design for desktop and mobile browsers
  • Built-in file manager for completed downloads

⚙ Configuration

  • Max 2 concurrent downloads (configurable)
  • 30 minute download timeout protection
  • Up to 50 videos for playlist merge
  • 24-hour auto-cleanup for downloaded files
  • Electron desktop app with portable option
